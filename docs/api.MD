# Endpoints da API



### Auth

* POST /auth/login

### Pesquisas

* POST /pesquisas
* GET /pesquisas
* GET /pesquisas/:id
* PATCH /pesquisas/:id
* PATCH /pesquisas/:id/publicar
* DELETE /pesquisas/:id

### Questões

* POST /questoes
* GET /questoes/:pesquisaId

### Respostas

* POST /respostas
* GET /respostas/relatorio/:pesquisaId

### Users (Admins)

* GET /users
* POST /users
* GET /users/:id
* PATCH /users/:id
* DELETE /users/:id
