# Flowbite React Frontend with Next.js

## Como rodar o frontend localmente

Com o repositório baixado, execute os seguintes passos:

```bash
cd frontend

cp .env.example .env

npm install

npm run dev
```

O link do app rodando localmente aparecerá no terminal.

> **Obs.:** para conectar o frontend com o backend é necessário primeiro rodar o backend em outro terminal antes de rodar o frontend. Também atualize as variáveis em .env caso seja necessário.
